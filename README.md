## ✨ Features

*   **🎯 FWOTD (First Win of the Day) Tracker:** Keep track of which accounts still need their daily bonus XP.

*   **⚡ Automated Setup Wizard:** Get started in seconds with our smart configuration tool that auto-detects your Riot Client.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ENG-AboSalama/VAM.git
   cd VAM
   ```

2. **Install dependencies:**
   Make sure you have Python 3.8+ installed. You can use the provided batch script or pip:
   ```bash
   # Using the batch script:
   install.bat
   
   # Or using pip directly:
   pip install -r requirements.txt
   ```

3. **Run VAM:**
   ```bash
   python main.py
   ```
   *Note: On your very first launch, the Setup Wizard will automatically guide you through the configuration process.*
